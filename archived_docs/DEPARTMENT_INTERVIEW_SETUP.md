## Department-Specific Content Requirements

### EE (Electrical Engineering)
**Topics:** circuits, power_systems, machines, electronics
**Beginner Questions:** Ohm's Law, KCL/KVL, AC vs DC, Series/Parallel, Transformers basics
**Intermediate Questions:** Three-phase systems, Motor types, Power factor, Control systems, Rectifiers
**Advanced Questions:** Protection relays, HVDC transmission, Power electronics, Renewable integration, Smart grids

### ME (Mechanical Engineering)
**Topics:** thermodynamics, fluids, materials, manufacturing
**Beginner Questions:** Laws of thermodynamics, Stress/strain, Fluid properties, Manufacturing processes
**Intermediate Questions:** Heat exchangers, Turbulent flow, Phase diagrams, CNC machining, CAD/CAM
**Advanced Questions:** Combustion engines, CFD analysis, Composite materials, Additive manufacturing, FEA

### CE (Civil Engineering)
**Topics:** structures, hydraulics, geotechnical, construction
**Beginner Questions:** Beam types, Concrete mix, Soil classification, Surveying basics, Load calculations
**Intermediate Questions:** Moment distribution, Open channel flow, Bearing capacity, Concrete design, Project scheduling
**Advanced Questions:** Seismic design, Dam engineering, Foundation design, Prestressed concrete, BIM integration

### EC (Electronics & Communication)
**Topics:** communications, microcontrollers, signals, networks
**Beginner Questions:** Modulation types, Microcontroller basics, Fourier transform, Network protocols, Boolean algebra
**Intermediate Questions:** AM/FM/PM, 8051 architecture, Filter design, TCP/IP, Op-amp circuits
**Advanced Questions:** MIMO systems, ARM processors, DSP algorithms, 5G technology, VLSI design

## Template Structure
Each department needs:
1. **preparation.html** - 16 video cards (4x4 grid) with department-specific content
2. **interview.html** - MCQ quiz system with 3 topics √ó 3 levels (9 question sets)
3. **ai-interview.html** - Copy of CS version with:
   - Department code (EE/ME/CE/EC)
   - Department-specific question banks
   - Same camera/emotion/WPM features
4. **ai-report.html** - Dark-themed report (copy CS version, update department ref)

## Data Storage
- Traditional interviews: `traditional_interviews/{dept}/{uid}/{sessionId}`
- AI interviews: `interviews/{uid}/{sessionId}` (with department field)
- Reports accessible via department-specific report.html pages
